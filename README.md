Auto Refresh Extension

一个轻量、高效的 Google Chrome 插件，帮你自动刷新指定网页。用户可以自定义刷新间隔，支持快捷键一键启动或停止刷新，操作简单直观。非常适合实时监控网页、抢购、查看动态更新或测试网页变化。插件界面简洁，占用资源低，安装即可使用，无需额外配置。通过智能脚本控制刷新行为，保证稳定性和流畅体验，是提升浏览效率的实用小工具。
